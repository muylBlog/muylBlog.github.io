Linux JDK安装及配置 (tar.gz版)

发表于 2016-07-10   |   分类于 Linux   |  

安装步骤

准备工作

  • 卸载openJDK
  • 检查jdk版本、卸载openjdk版本（jdk与openjdk的区别不在此赘述）
    先检查java –version、javac –version命令查看版本(如图一)
    Linux JDK安装及配置 (tar.gz版)
  • 经检查jdk版本为1.7.0.45；再输入rpm -qa|grep java
    如果没有列表显示，可直接看第4步
    卸载以上三个文件（需要root权限，登录root权限卸载）（如图三）
    rpm -e –nodeps java-1.7.0-openjdk-1.7.0.45-2.4.3.3.el6.x86_64
    rpm -e –nodeps java-1.6.0-openjdk-1.6.0.0-1.66.1.13.0.el6.x86_64
    rpm -e –nodeps tzdata-java-2013g-1.el6.noarch
  • 再次查看jdk信息

    安装OracleJDK

  • 下载jdk软件包到指定目录
  • 现在准备安装Oracle版的jdk。
  • 解压jdk，解压命令:tar -xvzf jdk-8u91-linux-x64.tar.gz
    jdk所在位置：/usr/local/jdk/jdk-8u91-linux-x64.tar.gz
  • 配置jdk环境变量
    在Linux系统中，需要在PATH中添加jdk的bin目录，创建CLASSPATH，添加jdk的lib目录。
    Jdk的主目录：/usr/local/jdk/jdk1.8.0_91
    jdk的bin目录：/usr/local/jdk/jdk1.8.0_91/bin
    jdk的lib目录：/usr/local/jdk/jdk1.8.0_91/lib
  • 编辑~/.bashrc文件，vi ~/.bashrc
    export JAVA_HOME=/home/codebrother/jdk/jdk1.8.0_91
    export JAVA_BIN=$JAVA_HOME/bin
    export JAVA_LIB=$JAVA_HOME/lib
    export CLASSPATH=.:$JAVA_LIB/tools.jar:$JAVA_LIB/dt.jar
    export PATH=$JAVA_BIN:$PATH
    • 使jdk环境变量生效，source ~/.bashrc
    • 查看jdk版本
      输入java -version、javac –version
